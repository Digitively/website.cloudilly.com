Features

More than just simple pubsub!

Cross Platform

Build cross-platform native mobile and web apps with our Web, iOS, and Android SDKs. You can also connect Cloudilly to your existing backend using our Hook SDK.

Realtime Database

Store and sync data with our NoSQL cloud database. Data is synced to all connected clients in realtime, and available when your app goes offline.

Intuitive methods

A succinct set of APIs that makes it easy to build realtime functionalities. Use cases include Group chat, on-demand Uber-like apps to complex MMORPG games.

Push Notifications

Increase user engagement with targeted push notifications. Send across devices and platforms with ease.

SSL connections
